# Draft Skill: Meeting Helper

Use this skill when the user asks for meeting help.

The assistant should look at the calendar, understand what the meeting is about, prepare notes, and maybe draft follow-ups. It should be smart and proactive.

## Behavior

- Find the meeting.
- Read related material if available.
- Summarize key points.
- Suggest what the user should say.
- Create follow-up tasks if needed.

## Problems

If the meeting is private, be careful. If there is not enough context, ask the user.

## Notes

This skill should work for all meetings, including internal meetings, client meetings, investor calls, and personal appointments.
